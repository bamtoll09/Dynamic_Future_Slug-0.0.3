    class RGEngine::Components::AnimationRenderer

    Rendering animations

  • bool loop

    true일시 재생이 끝나고 다시 처음부터 재생한다.

  • bool pause

    true일시 일시정지한다.

  • float speed

    재생 속도를 지정한다. 디폴트는 1.0f이다.

  • float nowFrame

    현재 프레임이다. (int)로 형변환하여 인덱스를 얻어올 수 있다.

  • Assets::Texture* GetTexture(int index) const const Assets::Texture** GetTextures(void) const

    Push한 이미지를 얻어온다. GetTextures 사용시 모두 얻어온다.

  • int GetSize(void) const

    Push된 이미지(프레임)의 개수를 얻어온다.

  • void PushBackFrame(std::string path)

    이미지를 삽입한다.

  • void ReplaceFrame(int index, std::string path)

    이미 있는 프레임을 해제하고 그 자리에 새로운 이미지를 삽입한다.

  • void Stop(void)

    재생을 중지하고 처음으로 되돌린다.

  • void Reset(void) override

    Component를 처음 상태로 초기화한다.
