    class RGEngine::Components::AudioSource

    Play sound

  • bool IsPlaying(void) const

    재생중인지 얻어온다.

  • void Play(void);

    재생한다.

  • void Pause(void)

    일시정지한다.

  • void Resume(void)

    일시정지를 해제한다.

  • void Stop(void)

    재생을 중지하고 처음으로 되돌린다.

  • Assets::Sound* GetSound(void) const

    설정한 사운드를 얻어온다.

  • void SetSound(std::string path)

    사운드를 설정한다.

  • unsigned int GetPosition(void) const

    재생중인 사운드의 재생위치를 밀리초 단위로 얻어온다.

  • float GetVolume(void) const

    현재 볼륨을 얻어온다.

  • float GetFrequency(void) const

    현재 주파수를 얻어온다.

  • float GetPan(void) const

    현재 패닝을 얻어온다.

  • bool GetMute(void) const

    현재 음소거 상태인지 얻어온다.

  • void SetPosition(unsigned int position)

    사운드의 재생위치를 밀리초 단위로 설정한다.

  • void SetVolume(float volume)

    볼륨을 설정한다.

    0.0f ~ 1.0f 의 값을 가진다.

  • void SetFrequency(float frequency)

    주파수를 설정한다.

    Hz로, 스트림 사운드가 아닌 경우 음수가 될 수 있다.

  • void SetPan(float pan)

    패닝을 설정한다.

    -1.0f ~ 1.0f의 값을 가진다. (음수일수록 왼쪽, 양수일수록 오른쪽)

  • void SetMute(bool b)

    음소거를 설정한다.

  • void Reset(void) override

    Component를 처음 상태로 초기화한다.
