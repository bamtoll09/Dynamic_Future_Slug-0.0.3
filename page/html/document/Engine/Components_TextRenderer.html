    class RGEngine::Components::AnimationRenderer

    Rendering animations

  • std::string text

    출력할 텍스트를 설정한다.

  • Types::Rect sourceRect

    출력할 사각형 영역을 설정한다. format에 따라 무시될 수도 있다.

  • bool flagAutoRect

    true일시 sourceRect를 무시하고 자동으로 사각형 영역을 지정한다. 디폴트는 true이다.

  • DWORD format

    출력 할 때 적용되는 속성이다. DT_로 시작한다. 디폴트는 DT_LEFT | DT_TOP | DT_NOCLIP 이다. (참고링크)

  • Math::Color color

    텍스트의 컬러를 지정한다.

  • bool SetFont(std::string name, unsigned int size, unsigned int weight = FW_NORMAL, bool italic = false)

    폰트를 설정한다.(초기화이므로 반드시 호출해야 한다.)

    각각 폰트 파일 이름, 크기, 굵기, 기울임이다.

    폰트 파일 이름에 주의. 예를 들어 '맑은 고딕'을 할 때엔 제어판 -> 글꼴로 들어가 맑은고딕을 열면 맑은고딕 굵게, 가늘게 등으로 나뉘는데, 그 파일들 중 원하는 것을 오른쪽 클릭하고 '속성'을 누르면 폰트 파일 이름이 나온다. ttf는 굳이 붙이이 않아도 된다. 예)malgunbd.ttf

  • void ReleaseFont(void)

    설정한 폰트를 해제한다. (폰트, 크기, 굵기, 기울임을 바꾸려면 해제하고 재설정해야 한다.)

  • LPD3DXFONT GetDXFont(void) const

    DirectX 폰트 객체를 얻어온다.

  • std::string GetFontName(void) const

    설정한 폰트 이름을 얻어온다.

  • unsigned int GetFontSize(void) const

    설정한 폰트 크기를 얻어온다.

  • unsigned int GetFontWeight(void) const

    설정한 폰트 굵기를 얻어온다.

  • bool GetFontItalic(void) const

    폰트가 기울임 상태인지 얻어온다.

  • void Reset(void) override

    Component를 처음 상태로 초기화한다.
