    class RGEngine::Application

    Singleton, GUI Bundle

  • static Application* GetInstance(void)

    싱글톤 클래스의 객체를 얻어온다.

    #define RGApp (RGEngine::Application::GetInstance)

  • bool Initialize(const wchar_t *title, int width, int height, bool fullscreen)

    Graphic, Input, Time 객체를 초기화하고 윈도우 화면을 띄운다.

    wchar_t는 유니코드 문자열이므로 L"타이틀" 과 같이 작성한다.(유니코드 표기)

  • void Release(void)

    Graphic, Input, Time, 현재 Scene 객체를 해제한다.

  • Graphic* GetGraphic(void) const

    Graphic 객체를 얻어온다.

    #define RGGraphic (RGApp->GetGraphic())

  • Input* GetInput(void) const

    Input 객체를 얻어온다.

    #define RGInput (RGApp->GetInput())

  • Time* GetTime(void) const

    Time 객체를 얻어온다.

    #define RGTime (RGApp->GetTime())

  • void ChangeScene(Scene *scene)

    현재 Scene을 바꾼다. 기존의 Scene은 삭제된다.

  • Scene* GetScene(void) const

    현재 Scene을 얻어온다.

  • template<typename SceneType> SceneType* GetScene(void) const

    현재 Scene을 SceneType으로 형변환 하여 얻어온다.

  • void Update(void)

    현재 Scene의 프레임을 갱신한다.

  • void Render(void)

    현재 Scene을 렌더링한다.

  • void Exit(void)

    프로그램을 종료한다.

  • inline double deltaTime(void)

    멤버 메서드가 아니다.

    RGTime->GetElapsedTime()을 deltaTime()으로 쉽게 접근하게 한다.
