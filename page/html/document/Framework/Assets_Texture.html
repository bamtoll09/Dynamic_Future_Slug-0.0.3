    class RGEngine::Assets::Texture

    Texture Asset

  • Types::Rect sourceRect

    출력할 이미지 내부의 사각형을 설정한다. (디폴트는 이미지 전체)

  • Math::Color color

    출력할 이미지의 색을 조정한다. 알파값을 바꾸면 투명도가 조절된다.

  • bool flagAutoRect

    true일시 sourceRect를 무시하고 자동으로 이미지 전체가 출력된다. (디폴트는 true)

  • bool Load(std::string path)
    bool Load(std::string path, Math::Color colorkey)

    이미지 파일을 불러온다. colorkey 지정시 이미지에서 해당 color는 투명화된다.

    TextureManager에서 호출된다.(따로 호출할 일이 없다.)

  • void Release(void)

    이미지 파일을 해제한다.

    TextureManager에서 호출된다.(따로 호출할 일이 없다.)

  • std::string GetPath(void) const

    이미지 에셋 경로를 얻어온다.

  • Math::Color GetColorkey(void) const

    지정한 Colorkey를 얻어온다.

  • int GetWidth(void) const
    int GetHeight(void) const
    Types::Size GetSize(void) const

    가로 세로 크기를 얻어온다.

  • void Blt(void)

    이미지를 출력한다.(2D Transformation과 함께 사용한다. 2D Transformation은 GameObject가 Render되기 전에 수행된다.)
