    class RGEngine::GameObject

    GameObject Interface

  • std::string tag

    태그를 저장할 수 있는 문자열

  • std::string name

    오브젝트 이름을 저장할 수 있는 문자열

  • GameObject *parent

    부모 오브젝트를 설정. localPosition의 기준(0, 0)이 부모 오브젝트의 좌표가 된다(따라다닌다).

    디폴트값은 Push된 Scene의 카메라

  • Math::Matrix matrix

    2D Transformation을 위한 행렬

  • Math::Vector position

    localPosition. 오브젝트의 좌표이다.

  • Math::Vector center

    오브젝트의 중심점을 지정한다. (실제 출력되는 좌표는 position - center)이 된다.

  • Math::Vector scale

    스케일을 지정한다. 스프라이트 혹은 텍스트의 크기에 곱해진다.

  • Math::Vector rotationCenter

    회전 중심점을 지정한다.

  • Math::Vector scalingCenter

    스케일 중심점을 지정한다. 디폴트는 왼쪽 위.

  • float rotate

    회전값을 지정한다. (Radian 단위)

  • float scalingRotation

    확대 회전 요소를 지정. (Radian 단위)

    스케일 값의 가로와 세로가 다를 때 영향.

  • void SetEnabled(bool flag)

    사용 가능 / 불가능 상태를 설정한다. 불가능 상태일시 Update/Render 되지 않는다.

  • bool GetEnabled(void) const

    현재 사용 가능 / 불가능 상태를 얻어온다.

  • template<typename T> T* AttachComponent(void)

    Component를 부착한다. (ex : gameObject->AttachComponent<SpriteRenderer>())

    부착된 Component를 반환한다.

  • template<typename T> void DetachComponent(void)

    부착된 Component를 해제한다.

  • template<typename T> T* GetComponent(void)

    부착된 Component를 얻어온다.

  • void Destroy(void)

    자기 자신을 파괴한다. 씬에서도 해제된다. (현재 프레임이 끝나는 시간에 해제됨.)

  • bool GetDestroyed(void) const

    자기 자신이 Destroy되었을시 true를 반환한다. (Destroy를 호출하고 현재 프레임이 끝나기 전까지 유효)

  • virtual void OnEnabled(void)

    GameObject를 상속한 클래스에서 Override 하여 사용한다.

    현재 오브젝트가 사용 가능 상태가 되었을 시 호출된다.

  • virtual void OnDisabled(void)

    GameObject를 상속한 클래스에서 Override 하여 사용한다.

    현재 오브젝트가 사용 불가능 상태가 되었을 시 호출된다.

  • virtual void OnUpdate(void)

    GameObject를 상속한 클래스에서 Override 하여 사용한다.

    프레임갱신시 호출된다.

  • virtual void OnAfterUpdate(void)

    GameObject를 상속한 클래스에서 Override 하여 사용한다.

    모든 오브젝트의 프레임 갱신이 끝난 후 렌더링 전 호출된다.

  • virtual void OnRender(void)

    GameObject를 상속한 클래스에서 Override 하여 사용한다.

    렌더링시 호출된다.

  • virtual void OnAfterRender(void)

    GameObject를 상속한 클래스에서 Override 하여 사용한다.

    모든 오브젝트의 렌더링이 끝난 후 호출된다.

  • virtual void OnCollision(GameObject *col)

    GameObject를 상속한 클래스에서 Override 하여 사용한다.

    현재 오브젝트에 Collider가 부착되어 있을 시 다른 Collider와 충돌중일때 호출된다. (유니티의 OnCollisionStay)
