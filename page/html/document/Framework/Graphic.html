    class RGEngine::Graphic

    Window & DirectX Manager

  • bool CreateFrame(const wchar_t *title, int width, int height, bool fullscreen)

    윈도우를 생성한다. Application의 Initialize에서 호출된다.

  • bool ChangeResolution(int width, int height)

    해상도를 변경한다.

  • void Release(void)

    DirectX 객체를 해제하고 윈도우를 파괴한다.

  • void PushResetableComponent(ResetableComponentInterface *component)

    ResetableComponentInterface에서 상속받은 객체를 추가한다. 디바이스가 Reset될때 함께 Reset된다.

    프레임워크 내부에서 호출된다. (따로 사용할 일이 없다.)

  • void PopResetableComponent(ResetableComponentInterface *component)

    추가된 ResetableComponentInterface에서 상속받은 객체를 삭제한다.

    프레임워크 내부에서 호출된다. (따로 사용할 일이 없다.)

  • bool Process(void)

    Windows 운영체제에서 받은 메시지를 Dispatch한다.

    게임의 메인루프 안에서 프레임마다 호출되어야한다.

  • bool BeginScene(void)

    렌더링을 시작한다. 렌더링 하기 전 호출되어 성공하여야한다. (성공시 true를 반환한다.)

  • bool EndScene(void)

    렌더링을 종료한다.

  • bool ToggleFullscreen(void)

    전체화면일시 창모드로, 창모드일시 전체화면으로 전환한다.

  • bool Validate(void)

    윈도우에 종료 메시지가 도착하였는지, DirectX 디바이스가 유효한지 검사한다. 매 프레임 갱신 전 체크하여야 한다.

  • bool IsFullscreen(void) const

    현재 풀스크린 상태인지 검사한다.

  • void SetCaption(const wchar_t *title)

    윈도우 타이틀(캡션)을 수정한다.

  • void SetWindowPos(const Types::Point &p)
    void SetWindowPos(int x, int y)

    윈도우 좌표를 지정한다.

    해상도 변경시 프레임워크 내부에서 호출된다.

  • void SetWindowSize(const Types::Size &s)
    void SetWindowSize(int width, int height)

    윈도우 크기를 지정한다.

    해상도 변경시 프레임워크 내부에서 호출된다.

  • void SetMaxmizable(bool flag)

    윈도우의 최대화 가능 / 불가능을 설정한다.

  • HWND GetWindowHandle(void) const

    윈도우의 핸들(Windows에서 윈도우들이 가지고있는 고유 식별값)을 얻어온다.

  • std::wstring GetCaption(void) const

    윈도우의 타이틀을 유니코드 문자열로 얻어온다.

  • int GetWindowWidth(void) const
    int GetWindowHeight(void) const
    Types::Size GetWindowSize(void) const

    윈도우의 크기를 얻어온다. (프레임 포함)

  • int GetScreenWidth(void) const
    int GetScreenHeight(void) const
    Types::Size GetScreenSize(void) const

    클라이언트의 가로 크기를 얻어온다. (윈도우의 프레임을 제외한 실제 게임화면 크기)

  • int GetWindowX(void) const
    int GetWindowY(void) const
    Types::Point GetWindowPos(void) const

    윈도우의 좌표를 얻어온다.

  • bool GetMaximizable(void) const

    윈도우의 최대화 가능 / 불가능을 얻어온다.

  • LPDIRECT3DDEVICE9 GetDevice(void) const

    DirectX9 디바이스 객체를 얻어온다.

  • LPD3DXSPRITE GetSprite(void) const

    DirectX9 스프라이트 객체를 얻어온다.

  • void MinimizeWindow(void)

    윈도우를 최소화 상태로 만든다.

  • void MaxmizeWindow(void)

    윈도우가 최대화 상태로 만든다.

  • void FocusWindow(void)

    윈도우에 포커스를 설정한다.

  • void ForegroundWindow(void)

    윈도우를 맨 위로 올라오게 한다.

  • void ActiveWindow(void)

    윈도우를 활성화중인 윈도우로 설정한다.

  • bool IsMinimizeWindow(void) const

    윈도우가 최소화 상태인지 얻어온다

  • bool IsMaximizeWindow(void) const

    윈도우가 최대화 상태인지 얻어온다.

  • bool IsFocusWindow(void) const

    윈도우에 포커스가 설정되어있는지 얻어온다.

  • bool IsActiveWindow(void) const

    윈도우가 활성화된 윈도우인지 얻어온다.
