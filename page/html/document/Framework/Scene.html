    class RGEngine::Scene

    Scene Interface

  • GameObject* GetCamera(void) const

    지금 씬의 카메라 오브젝트를 얻어온다.

  • GameObject** GetGameObjects(void) const

    현재 씬에 Push 되어있는 모든 GameObject를 배열로 얻어온다.

  • int GetGameObjectSize(void) const

    현재 씬에 Push 되어있는 GameObject의 개수를 얻어온다.

  • void PushBackGameObject(GameObject *object)

    씬에 GameObject를 Push한다.

    먼저 Push한 오브젝트가 먼저 업데이트, 렌더링된다. (늦게 push 할수록 렌더링시 위로 온다.)

  • virtual void OnUpdate(void)

    Scene를 상속한 클래스에서 Override 하여 사용한다.

    프레임갱신시 호출된다.

  • virtual void OnAfterUpdate(void)

    Scene를 상속한 클래스에서 Override 하여 사용한다.

    모든 오브젝트의 프레임 갱신이 끝난 후 렌더링 전 호출된다.

  • virtual void OnRender(void)

    Scene를 상속한 클래스에서 Override 하여 사용한다.

    렌더링시 호출된다.

  • virtual void OnAfterRender(void)

    Scene를 상속한 클래스에서 Override 하여 사용한다.

    모든 오브젝트의 렌더링이 끝난 후 호출된다.
